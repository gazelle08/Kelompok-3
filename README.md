## Sistem Informasi Pemesanan tiket konser

Sistem informasi ini akan membantu pengguna dalam mempermudah pemesanan tiket konser melalui situs tikethits.com.

## Fitur
fitur yang terdapat pada sistem informasi ini:
- jadwal konser
- reservasi
- pembayaran
- menampilkan tiket
